    二次方程基本概念

    二次方程定義

    二次方程是形如 \( ax^2 + bx + c = 0 \) 的方程，其中 \( a \neq 0 \)。

    二次方程的解稱為「根」，可能有以下三種情況：

    情況 根的性質 判別式 (Δ) 圖像特徵
    情況一 兩個相異實根 Δ > 0 拋物線與x軸有兩個交點
    情況二 一個二重實根（等根） Δ = 0 拋物線與x軸相切
    情況三 沒有實根（有兩個共軛複數根） Δ < 0 拋物線與x軸沒有交點
    重要注意

    在二次方程 \( ax^2 + bx + c = 0 \) 中，係數 \( a \) 絕對不能為零。如果 \( a = 0 \)，方程就變成一次方程，不再是二次方程。

    二次方程標準形式

    二次方程必須先化為標準形式 \( ax^2 + bx + c = 0 \) 才能使用公式法求解。

    \[ ax^2 + bx + c = 0 \]

    其中：

    • \( a \) 是二次項係數 (\( a \neq 0 \))
    • \( b \) 是一次項係數
    • \( c \) 是常數項
    例子：化為標準形式

    將方程 \( 2x(x-3) = 5 \) 化為標準形式：

    \[ 2x(x-3) = 5 \] \[ 2x^2 - 6x = 5 \] \[ 2x^2 - 6x - 5 = 0 \]

    此時 \( a = 2 \), \( b = -6 \), \( c = -5 \)。

    二次方程歷史背景

    二次方程的解法可以追溯到古代巴比倫、埃及、希臘、中國和印度等文明。

    • 巴比倫（公元前2000年）：已有解決二次方程問題的記錄
    • 古希臘：歐幾里得在《幾何原本》中給出幾何解法
    • 古印度：婆羅摩笈多（公元7世紀）給出二次方程的一般解法
    • 阿拉伯：花拉子米（公元9世紀）在其著作《代數學》中系統研究二次方程

    現代二次方程公式 \( x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \) 是由16世紀歐洲數學家發展完善的。

    解二次方程的三種方法

    方法一：十字相乘法

    適用於容易分解的二次方程。基本思想是將二次三項式分解為兩個一次因式的乘積。

    \[ ax^2 + bx + c = (px + q)(rx + s) \]

    其中 \( p \times r = a \), \( q \times s = c \), 且 \( p \times s + q \times r = b \)。

    示例：\( x^2 - 5x + 6 = 0 \)
    \[ x^2 - 5x + 6 = 0 \] \[ (x-2)(x-3) = 0 \] \[ x-2 = 0 \quad \text{或} \quad x-3 = 0 \] \[ x = 2 \quad \text{或} \quad x = 3 \]

    驗算：\( 2^2 - 5 \times 2 + 6 = 4 - 10 + 6 = 0 \)，\( 3^2 - 5 \times 3 + 6 = 9 - 15 + 6 = 0 \)

    十字相乘法要點

    1. 先分解二次項係數 \( a \) 和常數項 \( c \)

    2. 交叉相乘後的和應等於一次項係數 \( b \)

    3. 當 \( a \neq 1 \) 時，可能需要嘗試多種組合

    方法二：取平方根法

    適用於形如 \( (x-h)^2 = k \) 的二次方程，其中 \( k \geq 0 \)。

    \[ (x-h)^2 = k \] \[ x-h = \pm \sqrt{k} \] \[ x = h \pm \sqrt{k} \]
    示例：\( (x-4)^2 = 64 \)
    \[ (x-4)^2 = 64 \] \[ x-4 = \pm \sqrt{64} \] \[ x-4 = \pm 8 \] \[ x = 4 + 8 \quad \text{或} \quad x = 4 - 8 \] \[ x = 12 \quad \text{或} \quad x = -4 \]
    取平方根法步驟
    1. 確保方程為 \( (x-h)^2 = k \) 形式
    2. 兩邊取平方根，記得加 ± 號
    3. 解出 x 的兩個值

    方法三：公式法

    適用於所有二次方程，使用二次方程公式：

    \[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]
    示例：\( 2x^2 - 7x + 5 = 0 \)
    \[ a = 2, \quad b = -7, \quad c = 5 \] \[ x = \frac{-(-7) \pm \sqrt{(-7)^2 - 4 \times 2 \times 5}}{2 \times 2} \] \[ x = \frac{7 \pm \sqrt{49 - 40}}{4} \] \[ x = \frac{7 \pm \sqrt{9}}{4} \] \[ x = \frac{7 \pm 3}{4} \] \[ x = \frac{7 + 3}{4} = \frac{10}{4} = 2.5 \quad \text{或} \quad x = \frac{7 - 3}{4} = \frac{4}{4} = 1 \]
  • 步驟1：確定係數 \( a, b, c \) 的值
  • 步驟2：計算判別式 \( \Delta = b^2 - 4ac \)
  • 步驟3：將數值代入公式 \( x = \frac{-b \pm \sqrt{\Delta}}{2a} \)
  • 步驟4：化簡得到兩個解（如果 Δ ≥ 0）

    • 三種方法比較

    方法 適用情況 優點 缺點
    十字相乘法 係數簡單，容易分解 計算快，無需公式 不適用於所有方程
    取平方根法 方程為 \( (x-h)^2 = k \) 形式 直接簡單 需要先配方成完全平方
    公式法 所有二次方程 通用，一定能求解 計算較複雜，需記憶公式
    選擇建議

    1. 先嘗試十字相乘法（如果係數簡單）

    2. 如果方程已是 \( (x-h)^2 = k \) 形式，使用取平方根法

    3. 其他情況一律使用公式法

    二次方程兩根關係

    韋達定理 (Vieta's Formulas)

    若二次方程 \( ax^2 + bx + c = 0 \) 的兩根為 \( \alpha \) 和 \( \beta \)，則：

    \[ \alpha + \beta = -\frac{b}{a} \]
    \[ \alpha \beta = \frac{c}{a} \]

    這兩個關係稱為韋達定理，由法國數學家弗朗索瓦·韋達發現。

    示例驗證

    方程 \( 2x^2 - 7x + 5 = 0 \) 的根為 \( \alpha = 2.5 \) 和 \( \beta = 1 \)

    \[ \alpha + \beta = 2.5 + 1 = 3.5 \] \[ -\frac{b}{a} = -\frac{(-7)}{2} = \frac{7}{2} = 3.5 \] \[ \alpha \beta = 2.5 \times 1 = 2.5 \] \[ \frac{c}{a} = \frac{5}{2} = 2.5 \]

    驗證無誤！

    韋達定理的應用

    應用一：已知一根求另一根

    若已知方程 \( 2x^2 - 7x + 5 = 0 \) 的一個根是 1，求另一個根。

    \[ \alpha + \beta = -\frac{b}{a} = \frac{7}{2} \] \[ 1 + \beta = \frac{7}{2} \] \[ \beta = \frac{7}{2} - 1 = \frac{5}{2} = 2.5 \]

    應用二：建立二次方程

    已知兩根為 3 和 -2，建立二次方程。

    \[ \alpha + \beta = 3 + (-2) = 1 \] \[ \alpha \beta = 3 \times (-2) = -6 \] \[ \text{設 } a = 1 \text{，則 } b = -(\alpha + \beta) = -1, \quad c = \alpha \beta = -6 \] \[ \text{方程為： } x^2 - x - 6 = 0 \]

    應用三：求根的對稱式值

    已知 \( \alpha, \beta \) 是 \( x^2 - 5x + 3 = 0 \) 的根，求 \( \alpha^2 + \beta^2 \)。

    \[ \alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta \] \[ = (5)^2 - 2 \times 3 = 25 - 6 = 19 \]

    兩根關係推廣

    除了兩根之和與積，還可以推導出其他對稱式：

    \[ \alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta \]
    \[ \frac{1}{\alpha} + \frac{1}{\beta} = \frac{\alpha + \beta}{\alpha\beta} \]
    \[ \alpha^3 + \beta^3 = (\alpha + \beta)^3 - 3\alpha\beta(\alpha + \beta) \]
    \[ (\alpha - \beta)^2 = (\alpha + \beta)^2 - 4\alpha\beta \]
    記憶技巧

    這些公式都可以從 \( \alpha + \beta \) 和 \( \alpha\beta \) 推導出來，無需死記硬背。

    例如：\( \alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta \) 是從 \( (a+b)^2 = a^2 + 2ab + b^2 \) 變形而來。

    二次函數圖像特徵

    拋物線基本概念

    二次函數 \( y = ax^2 + bx + c \) (\( a \neq 0 \)) 的圖像是一條拋物線。

    拋物線圖示例：\( y = 2x^2 - 7x + 5 \)

    拋物線的重要特徵：

    • 開口方向：由 \( a \) 的正負決定，\( a > 0 \) 開口向上，\( a < 0 \) 開口向下
    • 頂點：拋物線的最高點或最低點，坐標為 \( \left( -\frac{b}{2a}, f\left(-\frac{b}{2a}\right) \right) \)
    • 對稱軸：通過頂點的垂直線，方程為 \( x = -\frac{b}{2a} \)
    • y截距：當 \( x = 0 \) 時，\( y = c \)
    • x截距：方程 \( ax^2 + bx + c = 0 \) 的實根，即圖像與x軸的交點

    開口方向與係數關係

    係數 \( a \) 的值 開口方向 頂點性質 極值 圖像示例
    \( a > 0 \) 向上 最低點 有極小值 ∪ 形
    \( a < 0 \) 向下 最高點 有極大值 ∩ 形
    注意

    係數 \( a \) 的絕對值 \( |a| \) 決定拋物線的「寬窄」：

    • \( |a| \) 越大，拋物線越窄
    • \( |a| \) 越小，拋物線越寬
    • \( |a| = 1 \) 時為標準拋物線 \( y = x^2 \)

    尋找拋物線特徵的步驟

  • 步驟1：確定開口方向 - 檢查係數 \( a \) 的正負
  • 步驟2：計算頂點坐標 - 使用公式 \( \left( -\frac{b}{2a}, f\left(-\frac{b}{2a}\right) \right) \)
  • 步驟3：確定對稱軸 - 方程為 \( x = -\frac{b}{2a} \)
  • 步驟4：求y截距 - 計算 \( f(0) = c \)
  • 步驟5：求x截距 - 解方程 \( ax^2 + bx + c = 0 \)
  • 步驟6：繪製圖像 - 根據以上信息繪製拋物線
    • 示例：分析 \( y = -x^2 + 4x + 5 \)

    1. 開口方向：\( a = -1 < 0 \)，開口向下

    2. 頂點坐標：\( x = -\frac{4}{2(-1)} = 2 \)，\( y = -(2)^2 + 4(2) + 5 = 9 \)，頂點為 (2, 9)

    3. 對稱軸：\( x = 2 \)

    4. y截距：\( f(0) = 5 \)，點 (0, 5)

    5. x截距：解 \( -x^2 + 4x + 5 = 0 \) 得 \( x = -1 \) 或 \( x = 5 \)

    一般式與頂點式轉換

    二次函數的三種形式

    形式 表達式 優點 缺點
    一般式 \( y = ax^2 + bx + c \) 直接顯示y截距 \( c \) 頂點不明顯
    頂點式 \( y = a(x-h)^2 + k \) 直接顯示頂點 \( (h, k) \) 需通過配方得到
    因式式 \( y = a(x-r_1)(x-r_2) \) 直接顯示x截距 \( r_1, r_2 \) 需方程有實根

    配方法：一般式轉頂點式

    配方法是將一般式轉換為頂點式的關鍵技術。

    \[ y = ax^2 + bx + c \]

    配方法步驟：

  • 步驟1：提取二次項係數 \( a \)（如果 \( a \neq 1 \)）
  • 步驟2：將括號內配成完全平方
  • 步驟3：化簡為頂點式 \( y = a(x-h)^2 + k \)
    • 示例：\( y = 2x^2 - 7x + 5 \)
    \[ y = 2x^2 - 7x + 5 \] \[ y = 2\left(x^2 - \frac{7}{2}x\right) + 5 \] \[ y = 2\left[x^2 - \frac{7}{2}x + \left(\frac{7}{4}\right)^2 - \left(\frac{7}{4}\right)^2\right] + 5 \] \[ y = 2\left[\left(x - \frac{7}{4}\right)^2 - \frac{49}{16}\right] + 5 \] \[ y = 2\left(x - \frac{7}{4}\right)^2 - \frac{49}{8} + 5 \] \[ y = 2\left(x - \frac{7}{4}\right)^2 - \frac{9}{8} \]

    所以頂點為 \( \left( \frac{7}{4}, -\frac{9}{8} \right) \)，對稱軸為 \( x = \frac{7}{4} \)。

    頂點式轉一般式

    將頂點式展開即可得到一般式：

    \[ y = a(x-h)^2 + k \] \[ y = a(x^2 - 2hx + h^2) + k \] \[ y = ax^2 - 2ahx + ah^2 + k \]

    對比 \( y = ax^2 + bx + c \)，可得：

    \[ a = a, \quad b = -2ah, \quad c = ah^2 + k \]
    示例：\( y = 3(x-2)^2 + 4 \)
    \[ y = 3(x-2)^2 + 4 \] \[ y = 3(x^2 - 4x + 4) + 4 \] \[ y = 3x^2 - 12x + 12 + 4 \] \[ y = 3x^2 - 12x + 16 \]

    所以一般式為 \( y = 3x^2 - 12x + 16 \)。

    配方法的記憶口訣

    配方法四步口訣

    一提取：提取二次項係數 \( a \)

    二配方：括號內加上減去一次項係數一半的平方

    三整理：將括號內寫成完全平方形式

    四化簡：化簡常數項得到頂點式

    // 配方法算法示例 (JavaScript)
function completeSquare(a, b, c) {
    // y = ax² + bx + c 轉為 y = a(x-h)² + k
    const h = -b / (2 * a);
    const k = c - a * h * h;
    return { h, k };
}

// 示例：y = 2x² - 7x + 5
const result = completeSquare(2, -7, 5);
console.log(`頂點: (${result.h}, ${result.k})`);
console.log(`頂點式: y = 2(x - ${result.h})² + ${result.k}`);

    直線方程

    直線方程形式

    1. 一般式

    \[ ax + by + c = 0 \]

    其中：

    • 斜率 \( m = -\frac{a}{b} \)（當 \( b \neq 0 \)）
    • x截距 \( = -\frac{c}{a} \)（當 \( a \neq 0 \)）
    • y截距 \( = -\frac{c}{b} \)（當 \( b \neq 0 \)）

    2. 斜截式

    \[ y = mx + c \]

    其中 \( m \) 為斜率，\( c \) 為y截距。

    3. 點斜式

    \[ y - y_1 = m(x - x_1) \]

    其中 \( m \) 為斜率，\( (x_1, y_1) \) 為直線上一點。

    4. 兩點式

    \[ \frac{y - y_1}{y_2 - y_1} = \frac{x - x_1}{x_2 - x_1} \]

    其中 \( (x_1, y_1) \) 和 \( (x_2, y_2) \) 為直線上兩點。

    直線的平行與垂直條件

    設兩直線斜率分別為 \( m_1 \) 和 \( m_2 \)：

    \[ \text{平行：} m_1 = m_2 \]
    \[ \text{垂直：} m_1 \times m_2 = -1 \]
    示例1：求平行直線

    直線 \( L_1: y = 2x + 3 \)，求通過點 (4, 5) 且平行於 \( L_1 \) 的直線。

    \[ m = 2 \quad (\text{平行，所以斜率相同}) \] \[ y - 5 = 2(x - 4) \] \[ y = 2x - 8 + 5 \] \[ y = 2x - 3 \]
    示例2：求垂直直線

    直線 \( L_1: y = -\frac{1}{3}x + 2 \)，求通過點 (-4, 1) 且垂直於 \( L_1 \) 的直線。

    \[ m_1 = -\frac{1}{3} \] \[ m_2 = -1 \div m_1 = -1 \div \left(-\frac{1}{3}\right) = 3 \] \[ y - 1 = 3[x - (-4)] \] \[ y = 3x + 12 + 1 \] \[ y = 3x + 13 \]

    求直線方程的方法總結

  • 情況1：已知一點和斜率 - 使用點斜式 \( y - y_1 = m(x - x_1) \)
  • 情況2：已知兩點 - 先求斜率 \( m = \frac{y_2 - y_1}{x_2 - x_1} \)，再用點斜式
  • 情況3：已知斜率和y截距 - 直接使用斜截式 \( y = mx + c \)
  • 情況4：已知x截距和y截距 - 使用截距式 \( \frac{x}{a} + \frac{y}{b} = 1 \)
  • 情況5：已知平行或垂直關係 - 先確定斜率，再用點斜式
    • 特殊情況注意

    1. 垂直於x軸的直線：斜率不存在，方程為 \( x = k \)

    2. 平行於x軸的直線：斜率為0，方程為 \( y = k \)

    3. 通過原點的直線：y截距為0，方程為 \( y = mx \)

    圖解法解二次方程

    圖解法原理

    二次函數 \( y = ax^2 + bx + c \) 的圖像與x軸的交點（x截距）就是二次方程 \( ax^2 + bx + c = 0 \) 的實根。

    圖解示例：\( y = x^2 + x - 4 \)

    從圖像可見，拋物線與x軸相交於 -2.6 和 1.6（準確至一位小數），所以方程 \( x^2 + x - 4 = 0 \) 的實根是 -2.6 和 1.6。

    圖解法優點

    1. 直觀顯示解的數量

    2. 顯示解的近似值

    3. 可觀察函數的整體行為

    圖解法步驟

  • 步驟1：建立坐標系 - 繪製x軸和y軸，標示適當刻度
  • 步驟2：計算關鍵點 - 計算頂點、y截距，可能的話計算x截距
  • 步驟3：繪製拋物線 - 根據關鍵點描繪拋物線圖像
  • 步驟4：找出交點 - 找出拋物線與x軸的交點
  • 步驟5：讀取解 - 交點的x坐標就是方程的解
    • 圖解法限制

    1. 只能得到近似解，精度有限

    2. 當解是無理數時，無法得到精確值

    3. 當拋物線與x軸沒有交點時，無法用圖解法求實根

    4. 作圖誤差會影響結果準確性

    圖解法與代數法比較

    方法 優點 缺點 適用情況
    圖解法 直觀，可顯示函數整體行為 精度低，只能得近似解 快速估計解，驗證代數解
    代數法 精確，能得到準確解 不夠直觀，計算可能複雜 需要精確解，考試答題
    建議使用方法

    1. 先用圖解法估計解的範圍和數量

    2. 再用代數法求精確解

    3. 最後用圖解法驗證解的合理性

    判別式與根的性質

    判別式定義

    對於二次方程 \( ax^2 + bx + c = 0 \)，判別式為：

    \[ \Delta = b^2 - 4ac \]

    判別式的值決定二次方程的根性質。

    判別式的幾何意義

    判別式 \( \Delta \) 的符號決定拋物線 \( y = ax^2 + bx + c \) 與x軸的交點數目：

    • \( \Delta > 0 \)：兩個交點（兩個相異實根）
    • \( \Delta = 0 \)：一個交點（相切，二重實根）
    • \( \Delta < 0 \)：沒有交點（沒有實根）

    判別式與根的情況

    情況 判別式 (Δ) 根的性質 因式分解 圖像的x截距數目
    情況一 Δ > 0 兩個不相等實根 可分解為兩個不同實係數一次因式 2個
    情況二 Δ = 0 一個二重實根（等根） 可分解為完全平方式 1個（與x軸相切）
    情況三 Δ < 0 沒有實根（有兩個共軛複數根） 不能分解為實係數一次因式 0個
    示例：\( x^2 - 4x - k = -5 \)

    若方程沒有實根，求 \( k \) 值的範圍。

    \[ x^2 - 4x - k + 5 = 0 \] \[ \Delta = (-4)^2 - 4(1)(-k+5) < 0 \] \[ 16 + 4k - 20 < 0 \] \[ 4k - 4 < 0 \] \[ 4k < 4 \] \[ k < 1 \]

    判別式的應用

    應用一：判斷根的情況

    不解方程，判斷 \( 2x^2 - 3x + 5 = 0 \) 的根的情況。

    \[ \Delta = (-3)^2 - 4(2)(5) = 9 - 40 = -31 < 0 \]

    因為 \( \Delta < 0 \)，所以方程沒有實根。

    應用二：求參數範圍

    若 \( y = kx^2 - 6x + k \) 的圖像與x軸只接觸於一點，求 \( k \) 值。

    \[ \Delta = (-6)^2 - 4(k)(k) = 0 \] \[ 36 - 4k^2 = 0 \] \[ 4k^2 = 36 \] \[ k^2 = 9 \] \[ k = \pm 3 \]

    應用三：證明根的情況

    證明方程 \( x^2 - 2mx + (m^2 - 1) = 0 \) 有兩個相異實根。

    \[ \Delta = (-2m)^2 - 4(1)(m^2 - 1) \] \[ = 4m^2 - 4(m^2 - 1) \] \[ = 4m^2 - 4m^2 + 4 \] \[ = 4 > 0 \]

    因為 \( \Delta > 0 \)，所以方程有兩個相異實根。

    判別式與複數根

    當 \( \Delta < 0 \) 時，二次方程沒有實根，但有兩個共軛複數根：

    \[ x = \frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-b \pm i\sqrt{-\Delta}}{2a} \]

    其中 \( i = \sqrt{-1} \) 是虛數單位。

    示例：\( x^2 + 2x + 5 = 0 \)
    \[ \Delta = 2^2 - 4(1)(5) = 4 - 20 = -16 \] \[ x = \frac{-2 \pm \sqrt{-16}}{2} = \frac{-2 \pm 4i}{2} = -1 \pm 2i \]

    所以方程的兩個共軛複數根是 \( -1 + 2i \) 和 \( -1 - 2i \)。

    複數

    複數定義

    複數是形如 \( a + bi \) 的數，其中 \( a \) 和 \( b \) 都是實數，\( i \) 是虛數單位，滿足 \( i^2 = -1 \)。

    \[ i^2 = -1 \]

    其中：

    • \( a \) 稱為實部 (Real part)
    • \( b \) 稱為虛部 (Imaginary part)
    注意

    1. 當 \( b = 0 \) 時，複數 \( a + 0i = a \) 就是實數

    2. 當 \( a = 0 \) 且 \( b \neq 0 \) 時，複數 \( 0 + bi = bi \) 稱為純虛數

    3. 所有實數都是複數（虛部為0的複數）

    複數四則運算

    1. 加法

    \[ (a + bi) + (c + di) = (a + c) + (b + d)i \]

    2. 減法

    \[ (a + bi) - (c + di) = (a - c) + (b - d)i \]

    3. 乘法

    \[ (a + bi)(c + di) = (ac - bd) + (ad + bc)i \]

    例：\( (2 - i)(3 - i) = 6 - 2i - 3i + i^2 = 6 - 5i - 1 = 5 - 5i \)

    4. 除法

    需要將分母實數化（乘以共軛複數）：

    \[ \frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{c^2 + d^2} \]

    例：\( \frac{1 + 2i}{3 - 4i} = \frac{(1+2i)(3+4i)}{3^2 + 4^2} = \frac{3+4i+6i+8i^2}{25} = \frac{3+10i-8}{25} = \frac{-5+10i}{25} = -\frac{1}{5} + \frac{2}{5}i \)

    共軛複數

    複數 \( z = a + bi \) 的共軛複數記作 \( \overline{z} = a - bi \)。

    \[ \text{若 } z = a + bi \text{，則 } \overline{z} = a - bi \]

    共軛複數的性質：

    \[ z + \overline{z} = 2a \quad (\text{實數}) \]
    \[ z - \overline{z} = 2bi \quad (\text{純虛數}) \]
    \[ z \cdot \overline{z} = a^2 + b^2 \quad (\text{實數，且 } \geq 0) \]
    共軛複數的應用

    1. 實數化分母：\( \frac{1}{a+bi} = \frac{a-bi}{a^2+b^2} \)

    2. 解二次方程：當 \( \Delta < 0 \) 時，兩個根互為共軛複數

    3. 簡化計算：某些複數運算中使用共軛可以簡化

    複數運算示例

    示例1：加法

    計算 \( (-1 + 4i) + (3 - 9i) \)：

    \[ (-1 + 4i) + (3 - 9i) = (-1+3) + (4-9)i = 2 - 5i \]
    示例2：乘法

    計算 \( (2 - i)(3 - i) \)：

    \[ (2 - i)(3 - i) = 6 - 2i - 3i + i^2 = 6 - 5i - 1 = 5 - 5i \]
    示例3：除法

    計算 \( \frac{3+i}{1-2i} \)：

    \[ \frac{3+i}{1-2i} = \frac{(3+i)(1+2i)}{1^2+2^2} = \frac{3+6i+i+2i^2}{5} = \frac{3+7i-2}{5} = \frac{1+7i}{5} = \frac{1}{5} + \frac{7}{5}i \]

    盈利與虧蝕

    基本概念

    術語 定義 公式 示例
    成本 (Cost) 商品的進貨成本 C $300
    標價 (Marked Price) 商品的原定價格 M $500
    售價 (Selling Price) 商品實際出售的價格 S $450
    折扣 (Discount) 標價減少的百分比 d% 10% off
    盈利 (Profit) 售價高於成本的部分 P = S - C $150
    虧蝕 (Loss) 成本高於售價的部分 L = C - S -

    重要公式

    售價 = 標價 × (1 - 折扣率)
    盈利 = 售價 - 成本
    虧蝕 = 成本 - 售價
    盈利百分率 = \( \frac{\text{盈利}}{\text{成本}} \times 100\% \)
    虧蝕百分率 = \( \frac{\text{虧蝕}}{\text{成本}} \times 100\% \)
    折扣百分率 = \( \frac{\text{標價 - 售價}}{\text{標價}} \times 100\% \)

    應用例子

    示例1：手袋問題

    某手袋的標價為 $500。現該手袋以其標價九折售出。

    (a) 求該手袋的售價。

    (b) 若該手袋獲利 $90，求該手袋的盈利百分率。

    (a) 售價 = 標價 × 折扣 = $500 × 90% = $450
    (b) 成本 = 售價 - 盈利 = $450 - $90 = $360
    盈利百分率 = \( \frac{\text{盈利}}{\text{成本}} \times 100\% = \frac{90}{360} \times 100\% = 25\% \)
    示例2：曲奇問題

    某禮品包內只有牛油曲奇及朱古力曲奇共 120 塊。該禮品包內有 5 個盒且每個盒均有相同數目的曲奇。在每個盒內，牛油曲奇均較朱古力曲奇多 6 塊。該禮品包內有多少塊朱古力曲奇？

    每盒曲奇數目：\( 120 \div 5 = 24 \) 塊
    設每盒朱古力曲奇有 \( y \) 塊，則牛油曲奇有 \( y + 6 \) 塊
    \( y + (y + 6) = 24 \)
    \( 2y + 6 = 24 \)
    \( 2y = 18 \)
    \( y = 9 \)
    全禮品包朱古力曲奇：\( 9 \times 5 = 45 \) 塊

    盈利虧蝕問題解題策略

  • 步驟1：明確已知量 - 識別題目中的成本、標價、售價、折扣、盈利/虧蝕
  • 步驟2：確定未知量 - 明確題目要求的是什麼
  • 步驟3：選擇適當公式 - 根據已知和未知量選擇公式
  • 步驟4：建立方程 - 必要時建立方程求解
  • 步驟5：計算並檢查 - 計算結果，檢查是否合理
    • 常見錯誤

    1. 混淆盈利百分率和折扣百分率

    2. 盈利百分率的基數是成本，折扣百分率的基數是標價

    3. 忘記將百分比轉換為小數進行計算

    4. 盈利和虧蝕的符號處理錯誤

    答案：\( \frac{m^9}{2n^{12}} \)

    因式分解

    (a) \( 4x^2 - 19xy - 5y^2 \)

    (b) \( 4x^2 - 19xy - 5y^2 + 5x - 25y \)

    解析：

    (a) 使用十字相乘法：

    \[ 4x^2 - 19xy - 5y^2 = (4x + y)(x - 5y) \]

    驗證：\( (4x+y)(x-5y) = 4x^2 - 20xy + xy - 5y^2 = 4x^2 - 19xy - 5y^2 \)

    (b) 分組分解法：

    \[ 4x^2 - 19xy - 5y^2 + 5x - 25y \] \[ = (4x^2 - 19xy - 5y^2) + (5x - 25y) \] \[ = (4x + y)(x - 5y) + 5(x - 5y) \] \[ = (x - 5y)[(4x + y) + 5] \] \[ = (x - 5y)(4x + y + 5) \]

    答案：(a) \( (4x+y)(x-5y) \) (b) \( (x-5y)(4x+y+5) \)

    解以下方程：

    (a) \( (x-4)^2 = 64 \)

    (b) \( \frac{1}{2}x^2 - 11x + 9 = 0 \)

    解析：

    (a) 使用取平方根法：

    \[ (x-4)^2 = 64 \] \[ x-4 = \pm \sqrt{64} \] \[ x-4 = \pm 8 \] \[ x = 4 + 8 \quad \text{或} \quad x = 4 - 8 \] \[ x = 12 \quad \text{或} \quad x = -4 \]

    (b) 使用公式法，先將方程乘以2：

    \[ \frac{1}{2}x^2 - 11x + 9 = 0 \] \[ x^2 - 22x + 18 = 0 \quad (\text{乘以2}) \] \[ x = \frac{22 \pm \sqrt{(-22)^2 - 4(1)(18)}}{2(1)} \] \[ = \frac{22 \pm \sqrt{484 - 72}}{2} \] \[ = \frac{22 \pm \sqrt{412}}{2} \] \[ = \frac{22 \pm 2\sqrt{103}}{2} \] \[ = 11 \pm \sqrt{103} \]

    答案：(a) \( x = 12 \) 或 \( x = -4 \) (b) \( x = 11 \pm \sqrt{103} \)

    考慮 \( f(x) = 3x^2 - 5 \) 及 \( g(x) = x + 1 \)。求下列各題的值。

    (a) \( f(1) + g(2) \)

    (b) \( f(-3) \times g(4) \)

    (c) \( f(5) \div g(-6) \)

    解析：

    (a) 先計算各函數值：

    \[ f(1) = 3(1)^2 - 5 = 3 - 5 = -2 \] \[ g(2) = 2 + 1 = 3 \] \[ f(1) + g(2) = -2 + 3 = 1 \]

    (b) 計算各函數值：

    \[ f(-3) = 3(-3)^2 - 5 = 3 \times 9 - 5 = 27 - 5 = 22 \] \[ g(4) = 4 + 1 = 5 \] \[ f(-3) \times g(4) = 22 \times 5 = 110 \]

    \[ f(5) = 3(5)^2 - 5 = 3 \times 25 - 5 = 75 - 5 = 70 \] \[ g(-6) = -6 + 1 = -5 \] \[ f(5) \div g(-6) = 70 \div (-5) = -14 \]

    解二次方程 \( x^2 - kx = 3 \)，其中 \( k \) 是常数。

    A. \( x = \frac{k \pm \sqrt{k^2 + 12}}{2} \)

    B. \( x = \frac{k \pm \sqrt{k^2 - 12}}{2} \)

    C. \( x = \frac{-k \pm \sqrt{k^2 + 12}}{2} \)

    D. \( x = \frac{-k \pm \sqrt{k^2 - 12}}{2} \)

    解析：
    \[ x^2 - kx = 3 \] \[ x^2 - kx - 3 = 0 \] \[ a = 1, \quad b = -k, \quad c = -3 \] \[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] \[ = \frac{-(-k) \pm \sqrt{(-k)^2 - 4(1)(-3)}}{2(1)} \] \[ = \frac{k \pm \sqrt{k^2 + 12}}{2} \]

    答案：A

    常見錯誤：忘記將方程化為標準形式，或符號計算錯誤。

    若二次方程 \( kx^2 - kx + 1 = 0 \) 有一個二重實根，則 \( k = \)

    A. 4

    B. -4

    C. 4 或 -4

    D. 0 或 4

    解析：

    B. \(-3\)

    解析：

    若 \( \alpha \) 及 \( \beta \) 是方程 \( x^2 - x - 4 = 0 \) 的根，下列哪個方程的根是 \( \alpha + 1 \) 及 \( \beta + 1 \)？

    A. \( x^2 - 3x - 2 = 0 \)

    B. \( x^2 - 3x + 2 = 0 \)

    C. \( x^2 + 3x - 2 = 0 \)

    D. \( x^2 + 3x + 2 = 0 \)

    解析：

    原方程 \( x^2 - x - 4 = 0 \) 的根為 \( \alpha, \beta \)

    \[ \alpha + \beta = 1, \quad \alpha\beta = -4 \]

    新方程的根為 \( \alpha+1, \beta+1 \)

    \[ (\alpha+1) + (\beta+1) = \alpha + \beta + 2 = 1 + 2 = 3 \] \[ (\alpha+1)(\beta+1) = \alpha\beta + \alpha + \beta + 1 = -4 + 1 + 1 = -2 \]

    所以新方程為：

    \[ x^2 - [(\alpha+1)+(\beta+1)]x + (\alpha+1)(\beta+1) = 0 \] \[ x^2 - 3x - 2 = 0 \]

    答案：A

    重點總結

    二次方程核心知識點

  • 二次方程定義：\( ax^2 + bx + c = 0 \)（\( a \neq 0 \)）
  • 解的公式：\( x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \)
  • 判別式：\( \Delta = b^2 - 4ac \)，決定根的情況
  • 兩根關係：\( \alpha + \beta = -\frac{b}{a} \)，\( \alpha\beta = \frac{c}{a} \)
  • 三種解法：十字相乘法、取平方根法、公式法

    • 二次函數圖像要點

  • 拋物線特徵：開口方向由 \( a \) 決定，頂點坐標 \( \left( -\frac{b}{2a}, f\left(-\frac{b}{2a}\right) \right) \)
  • 對稱軸：\( x = -\frac{b}{2a} \)
  • 截距：y截距為 \( c \)，x截距為方程 \( ax^2 + bx + c = 0 \) 的實根
  • 配方法：將一般式轉為頂點式 \( y = a(x-h)^2 + k \)
  • 圖解法：利用圖像解方程，得到近似解

    • 常見錯誤提醒

    二次方程常見錯誤

    4. 忘記公式中的 ± 號，只計算一個解

    5. 十字相乘法分解錯誤

    二次函數常見錯誤

    1. 混淆頂點坐標公式中的符號

    3. 判斷開口方向時只看 \( a \) 的絕對值
